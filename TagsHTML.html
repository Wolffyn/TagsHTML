HTML: O esqueleto da web

O que é HTML?

O HTML é usado para criar páginas da web, que são documentos escritos em formato hipertexto, ou seja, são criados através de etiquetas (tags) HTML para formar o esqueleto e toda a estrutura base do site. Com o HTML, é possível definir a estrutura e o conteúdo de uma página da web bem como os links para outras páginas ou recursos. Além disso, o HTML também permite a inclusão de multimedia, como imagens, áudio e vídeo, tornando as páginas da web mais atraentes e interactivas.

Tag: < ul > , < ol > e < li >

A tag < ul > e < ol > são para criar listas, sendo < ul > para criar listas não definidas e < ol > listas definidas.

Já a tag < li > é o elemento que entra dentro das listas, formando tópicos.

Com < ol >:

  2. Tópico
  3. Tópico

Tag: < blockquote >

A tag < blockquote > A tag blockquote especifica uma seção que é citada de outra fonte.

Exemplo:

Tag: < font >

A tag < font > foi usada em HTML 4 para especificar a fonte, o tamanho da fonte e a cor do texto. Entretanto não é suportada para o HTML5, pois hoje em dia utiliza-se a tecnologia CSS para modificar e personalizar fontes de texto.

Tag: < del >

A tag < del > define o texto que foi excluído de um documento. Os navegadores geralmente marcam uma linha através do texto excluído.

Exemplo:

As flores são verdes como folhas vermelhas como rosas.

Tag: < abbr >

A tag < abbr > A tag define uma abreviação ou um acrônimo, e ao encostar o ponteiro na palavra marcada mostrará o título completo.

Exemplo:

A OMS foi fundada em 1948.

Conclusão

o HTML evoluiu e se tornou uma tecnologia fundamental na construção da web moderna, permitindo a criação de páginas da web, aplicativos web e interfaces de usuário para aplicações mobilies e desktop.
